# Students Management

A comprehensive module to manage student information, attendance, leaves, skills, exams, and marks within Odoo.

## Key Features
- **Student Profiles:** Manage detailed student records.
- **Attendance Tracking:** Record and monitor daily attendance.
- **Academic Management:** Handle subjects, exams, and marks.
- **Leave Management:** Track and process student leave requests.
- **Bulk Operations:** Tools for bulk password resets.
- **Dashboards:** Integrated spreadsheet dashboards.

## Dependencies
- `base`
- `spreadsheet_dashboard`
