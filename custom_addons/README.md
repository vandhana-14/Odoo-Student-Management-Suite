# Odoo Student Management Suite

![Odoo Banner](https://www.odoo.com/web/image/website/1/logo/Odoo%20Logo?unique=465f8c8)

## Overview
This repository contains a collection of custom Odoo addons designed for comprehensive student management, covering everything from core student records and attendance to advanced dashboarding, examination management, and AI-assisted discussions.

## Modules
- **`Student_module`**: The foundational module for managing student profiles, departments, and basic skill tracking.
- **`students_app`**: An enhanced management suite extending student capabilities, including examination management, marks tracking, leave processing, and bulk password resets.
- **`students_dashboard`**: Provides visual analytics through spreadsheet-based dashboards to monitor student performance and institution metrics.
- **`ask_ai`**: An experimental module integrating AI features to facilitate better communication and support within the system.

## Features
- **Student Lifecycle Management**: Tracks students from enrollment to graduation.
- **Academic Tracking**: Comprehensive examination and marking systems.
- **Attendance & Leave**: Automated tracking for academic attendance and student leave requests.
- **Interactive Dashboards**: Real-time reporting via Odoo Spreadsheet integration.
- **AI Integration**: AI-driven tools to assist in academic discussions.

## Installation
1. Ensure you have an Odoo environment set up.
2. Clone this repository into your Odoo `addons_path` directory.
3. Restart your Odoo server.
4. Update your module list in Odoo.
5. Install the desired modules (e.g., `students_app`, `students_dashboard`).
